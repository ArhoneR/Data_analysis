# Data_analysis
В репозитории представлены учебные проекты.
# Основные инструменты и навыки:
* Языки: Python, SQL
* Анализ данных: библиотеки Pandas, NumPy, SciPy
* Визуализация: Matplotlib, Plotly, Seaborn
* Метрики юнит-экономики, когортный анализ
* А/В-тестирование
* Работа с гипотезами
